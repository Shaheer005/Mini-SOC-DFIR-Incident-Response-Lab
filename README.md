# Mini SOC & DFIR Incident Response Lab

## Overview
This project is a hands-on cybersecurity lab designed to simulate a small Security Operations Center (SOC) and Digital Forensics & Incident Response (DFIR) workflow.

The lab focuses on:
- Log generation
- Attack simulation
- Detection
- Investigation
- Incident reporting

All activities are conducted in an isolated virtual environment for educational purposes.

---

## Environment

| Machine | OS | Role | IP |
|------|----|----|----|
| Attacker | Kali Linux | Attack simulation | 192.168.2.120 |
| Victim | Ubuntu Linux | Log source | 192.168.2.104 |
| Optional Victim | Windows 10 | User endpoint | 192.168.2.150 |
